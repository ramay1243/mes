# Инструкция по запуску

## Шаг 1: Установка зависимостей

```bash
npm install
```

## Шаг 2: Настройка окружения

Создайте файл `.env` в корне проекта со следующим содержимым:

```
DATABASE_URL="file:./dev.db"
JWT_SECRET="your-secret-key-change-this-in-production-min-32-chars"
NEXT_PUBLIC_API_URL="http://localhost:3000"
```

## Шаг 3: Инициализация базы данных

```bash
npm run db:generate
npm run db:push
```

## Шаг 4: Запуск приложения

```bash
npm run dev
```

Откройте браузер и перейдите на http://localhost:3000

## Тестирование авторизации

1. Введите номер телефона (например: +7 999 123 45 67)
2. Нажмите "Отправить код"
3. **В консоли сервера** (где запущен `npm run dev`) вы увидите SMS код
4. Введите этот код в форму
5. Готово! Вы авторизованы

## Примечания

- В режиме разработки SMS коды выводятся в консоль сервера
- Для продакшена настройте реальную отправку SMS в файле `lib/sms.ts`
- База данных SQLite создается автоматически в файле `dev.db`
